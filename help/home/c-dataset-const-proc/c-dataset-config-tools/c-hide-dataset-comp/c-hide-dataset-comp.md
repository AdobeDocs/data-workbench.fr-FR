# Masquage des composants de jeu de données{#hiding-dataset-components}

Lorsque vous configurez votre jeu de données, vous pouvez masquer certaines dimensions étendues afin qu’elles ne s’affichent pas dans le menu des dimensions de l’outil de données.

Vous pouvez remplacer certains fichiers de configuration de jeux de données dans un profil que vous ne pouvez pas modifier ou ne souhaitez pas modifier. Les sections suivantes contiennent des instructions sur l&#39;exécution de ces tâches.
